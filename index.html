by Euccas

Being a software engineer, I keep learning and practicing my skills every single day. I build this ExpMap as a collection of the technical skills that I have cultivated, and the notes I took about them.

Before 2017, my notes were created and organized on Trello, which is a great tool for organizing and tracking projects and information. I started to use GitHub Wiki to keep my notes in 2017.

This bubble chart is built with D3.js, and you can view the source here on GitHub.
